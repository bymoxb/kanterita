# KANTERITA

## PUERTOS

Los puertos ocupados:

- Frontend (ReactJS): `3000`
- Backend (spring-boot): `8080`
- Base de datos: `5432`

## BACK

Iniciar Base de datos

```bash
cd back/
docker-compose up -d
```

Iniciar Proyecto Spring Boot

```bash
cd back/
./mvnw spring-boot:run
```

## FRONT

Iniciar aplicacion ReactJS

```bash
cd front/
npm i
npm start
```
